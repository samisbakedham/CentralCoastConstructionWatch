# Central Coast Construction Watch

A full-stack web application for tracking construction projects across Santa Barbara and San Luis Obispo Counties. Built with transparency and community engagement in mind.

## 🏗️ Features

### Public Access
- **Project Listings**: Browse all active construction projects with detailed information
- **Advanced Filtering**: Search by county, city, project type, status, and cost range
- **Project Details**: View comprehensive project information including timelines, costs, and residential unit details
- **Photo Timeline**: Track construction progress through timestamped photos
- **Responsive Design**: Optimized for desktop, tablet, and mobile viewing

### Admin Panel
- **Project Management**: Add, edit, and delete construction projects
- **Milestone Tracking**: Create and manage project milestones with descriptions and dates
- **Photo Management**: Upload and organize construction progress photos
- **Residential Details**: Track unit counts, types, and pricing for residential projects
- **Real-time Statistics**: Monitor active projects, total investment, and housing units

### Project Types Supported
- Residential developments
- Commercial buildings
- Infrastructure projects
- Mixed-use developments

## 🚀 Technology Stack

- **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: Session-based admin authentication
- **State Management**: TanStack Query (React Query)
- **Routing**: Wouter
- **Deployment**: Replit-ready
